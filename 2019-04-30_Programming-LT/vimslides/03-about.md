# 各種情報

  - 種類: Minecraft Mod
  - 使用言語: Python
  - 前提mod: [arpruss/raspberryjammod](https://github.com/arpruss/raspberryjammod)  -- PythonからMinecraftを弄れるようにするためのMod

# 詳細

## mcUI is
  - 3Dシェル環境
  - マイクラの中からCLIコマンドを使用できる
  - ディレクトリ構造を3Dで表示
  - 実験的なプロジェクト
  - つまる所暇人の遊び

## mcUI is not
  - UI framework (紛らわしすぎる)
  - 役に立つ何か


# 目的
  - ディレクトリ構造の3D化
  - CLIでもGUI(二次元)でもない、.新しいUIの模索.
